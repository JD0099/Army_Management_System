군대 관리 시스템
==============

##  프로젝트 개요

 디자인 패턴을 사용하여 군대 인원을 효율적으로 관리
 
 ## 기능
* 휴가,외박,외출 관리
* 상훈 관리
* 지휘관 명령사항 관리
* 보급품 관리
* 지시사항 관리
